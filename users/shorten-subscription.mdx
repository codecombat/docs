---
openapi: "PUT /users/{handle}/shorten-subscription"
description:
  "If the user already has a premium access up to a certain time, this
  shortens/revokes his/her premium access. If the ends is less than or equal to
  the current time, it revokes the subscription and sets the end date to be the
  current time, else it just shortens the subscription."
---

```python
url = `https://codecombat.com/api/users/${userID}/shorten-subscription`
json = { ends: new Date().toISOString() }
request.put({ url, json, auth }, (err, res) => {
  console.log(res.body.subscription.active) // false
})
```
